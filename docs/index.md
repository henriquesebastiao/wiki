# Início

Bem-vindo(a) à minha Wiki pessoal!

A ideia aqui é centralizar e disponibilizar informações que venho aprendendo ao longo dos anos, desde coisas triviais
que posso vir a me esquecer caso não exercite minha memória com frequência, até como utilizar determinadas ferramentas.
Vai ser tipo um *dump* da minha memória.
Isso é mais para meu uso pessoal do que para os outros, mas caso algum conteúdo dessa Wiki seja útil para você,
ficarei feliz em ter ajudado :).

Algumas coisas eu posto no meu [blog](https://henriquesebastiao.com/),
mas coisas mais técnicas e massantes como um passo a passo, que sirva como referência ficarão nessa Wiki.

O objetivo aqui não é ser perfeito, mas útil para mim e para todos.
